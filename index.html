community responsive, open-source

mdToolkit is a community driven, open-source suite of applications for authoring archival quality metadata to document data, projects, and collections.

flexible and adaptable solutions

Supports single users with a web browser, or organizations that wish to integrate metadata authoring tools into their workflows and infrastructure.

focus on content, not on standards

mdToolkit guides authors toward robust documentation without requiring knowledge of exchange standards.

manage standards transitions

mdToolkit separates metadata content management from exchange formats, allowing flexible transition workflows - write once, translate many times.

basic workflow

  • Author creates content using mdEditor
  • mdEditor writes the content as mdJSON
  • mdJSON Schemas provides validation to content
  • mdTranslator translates content in mdJSON format to an XML standard
